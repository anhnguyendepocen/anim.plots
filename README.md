anim.plots
==========

Simple animated plots for R. To install the bleeding-edge version:

```{splus}
install.packages("devtools") # if not already installed
library(devtools)
install_github("hughjonesd/anim.plots")
```

To install the latest stable release from CRAN:

```{splus}
install.packages("anim.plots")
```

For more information see the vignette, or [the presentation](http://hughjonesd.github.io/anim-plots-presentation.html).
